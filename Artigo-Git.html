Para que serve o Git, Github e Github Desktop

O que é Git?

É um Software de controle de versão, é um (VCS), é um software livre e gratúito

É uma ferramenta de versionamento distribuido não apenas de software, de código, mas serve também para outros tipos de versionamentos. Exemplo: Receita culinária, artigos entre outros, mas vamos falar aqui de códigos de software.

Principais Vantagens do Git

Controle de histórico, trabalho em equipe, ramificação de projeto, segurança, organização

O Git trabalha no lado cliente, na sua máquina, no seu computador, quando você

Imagine você trabalhando em equipe, no seu curso, na sua faculdade, no seu trabalho e estão fazendo um trabalho qualquer em equpe, imaginando que sua equipe tem quatro pessoas, cada pessoa trabalhando em um local diferente e não conseguem se comunicar por três dias, mas tem que entregar o trabalho em quatro dias.
